# Accessories Images

This folder contains images for traditional yurt accessories.

## Image Naming Convention

Each accessory should have an image named after its slug (ID):

- `shanyrak.jpg` - Shanyrak (crown of the yurt)
- `uyk.jpg` - Uyk (curved poles)
- `kerege.jpg` - Kerege (lattice walls)
- `esik.jpg` - Esik (wooden door)
- `tundik.jpg` - Tundik (felt cover for shanyrak)
- `uzik.jpg` - Uzik (felt mats for walls)
- `tuyrlyq.jpg` - Tuyrlyq (outer dome covering)
- `tuskiyz.jpg` - Tuskiyz (decorative felt panels)
- `ot-kilem.jpg` - Ot Kilem (central round carpet)
- `ak-baskur.jpg` - Ak Baskur (white decorative ribbon)
- `ak-kyzyl-baular.jpg` - Ak, Kyzyl Baular (white and red ropes)
- `ayir-bau.jpg` - Ayir Bau (spacer rope)
- `zhel-bau.jpg` - Zhel Bau (wind ropes)
- `shashak.jpg` - Shashak (decorative tassels)
- `shanyrak-ashekeyi.jpg` - Shanyrak Ashekeyi (decorative ornament)
- `chekhol.jpg` - Chekhol (protective cover)
- `bas-bel-arkandar.jpg` - Bas Bel Arkandar (main load-bearing ropes)
- `bakan.jpg` - Bakan (wooden stakes)
- `shalma-bau.jpg` - Shalma Bau (ropes for felt coverings)

## Image Requirements

- Format: JPEG or PNG
- Recommended size: 800x600px or larger
- Aspect ratio: 4:3 or 16:9
- File size: < 500KB for optimal loading

## Placeholder Images

Until actual photos are added, the system will use a default placeholder or the background image.

To add images:
1. Take or source high-quality photos of each accessory
2. Rename them according to the naming convention above
3. Place them in this folder
4. The images will automatically appear in the accessories modal and order views
