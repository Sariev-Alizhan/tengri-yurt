# Выкладка сайта на сервер и подключение домена

Сайт статический (HTML, CSS, JS), его можно разместить бесплатно или на купленном хостинге.

---

## PS Cloud Services (ps.kz) — твой вариант

Хостинг с панелью **Plesk**. Домен подключается к сайту в настройках хостинга.

### 1. Войти в панель

1. Зайди на **https://www.ps.kz** и войди в личный кабинет.
2. Открой свой **виртуальный хостинг** и зайди в панель управления **Plesk** (кнопка «Войти в Plesk» или ссылка из письма).

### 2. Загрузить файлы сайта

**Способ А: Файловый менеджер в Plesk**

1. В Plesk выбери **«Файлы»** (Files) или **«Диспетчер файлов»**.
2. Открой папку **`httpdocs`** (или **`public_html`** — как указано в подсказке на хостинге). В неё нужно загружать файлы сайта.
3. Загрузи с компьютера все файлы и папки из папки проекта Yurt:
   - `index.html`, `book-now.html`, `discover.html`
   - `styles.css`, `script.js`, `book-now.js`
   - папку **Images** целиком (со всеми картинками внутри)

**Способ Б: FTP**

1. В Plesk или в личном кабинете PS Cloud найди данные **FTP**: хост (например `ftp.твойдомен.kz` или IP), логин, пароль.
2. Подключись через FileZilla, WinSCP или «Этот компьютер → Подключить сетевой диск» (FTP).
3. Залей те же файлы в каталог **`httpdocs`** (или **`public_html`**).

Главное: **`index.html`** должен лежать прямо в **`httpdocs`**, а не в подпапке.

### 3. Подключить домен

- Если домен заказан **в PS Cloud** вместе с хостингом — он уже привязан, ничего дополнительно настраивать не нужно.
- Если домен куплен **у другого регистратора**: в Plesk в разделе **«Домены»** добавь домен к хостингу; в панели регистратора домена укажи **A-запись** на IP сервера PS Cloud (IP можно посмотреть в Plesk или в личном кабинете ps.kz).

### 4. HTTPS (SSL)

В Plesk: **«SSL/TLS-сертификаты»** → можно включить бесплатный сертификат **Let's Encrypt** для домена.

## Вариант 1: Бесплатный хостинг + ваш домен

Подойдёт, если куплен **только домен** (без хостинга). Сайт будет отдаваться с серверов Netlify/GitHub Pages, а в браузере будет открываться ваш домен.

### 1. Netlify (проще всего)

1. Зайдите на **https://netlify.com** и зарегистрируйтесь (можно через GitHub).
2. В кабинете: **Add new site → Deploy manually**.
3. Перетащите в окно браузера **всю папку проекта** (Yurt) — ту, где лежат `index.html`, `styles.css`, папка `Images` и т.д.  
   Либо: **Add new site → Import from Git** и выберите ваш репозиторий (тогда после каждого пуша сайт будет обновляться сам).
4. После загрузки Netlify даст адрес вида `random-name-123.netlify.app`.
5. **Подключение домена:**  
   **Site settings → Domain management → Add custom domain** → введите ваш домен (например `tengriyurt.com`).  
   Netlify покажет, какие записи DNS нужно создать.
6. В панели управления доменом (где покупали домен) создайте записи:
   - Для **корня домена** (tengriyurt.com):  
     Тип **A**, имя **@**, значение **75.2.60.5** (IP Netlify).
   - Для **www**:  
     Тип **CNAME**, имя **www**, значение **random-name-123.netlify.app** (ваш сайт на Netlify).

Через 5–60 минут домен начнёт открываться. В Netlify можно включить бесплатный HTTPS.

---

### 2. GitHub Pages + ваш домен

Если сайт уже в GitHub:

1. Репозиторий → **Settings → Pages**.
2. **Source:** Deploy from a branch.  
   **Branch:** main (или master), папка **/ (root)**. Сохраните.
3. Сайт появится по адресу `https://ВАШ_ЛОГИН.github.io/НАЗВАНИЕ_РЕПО/`.
4. **Свой домен:** в тех же **Pages** в поле **Custom domain** введите ваш домен и сохраните.
5. В настройках DNS у регистратора домена:
   - **A-записи** для корня: **185.199.108.153**, **185.199.109.153**, **185.199.110.153**, **185.199.111.153**
   - **CNAME** для **www**: значение **ВАШ_ЛОГИН.github.io**

## Вариант 2: Обычный хостинг (cPanel / FTP)

Если вы купили **домен + хостинг** (например, у reg.ru, Beget, Timeweb и т.п.):

1. В письме или панели хостинга найдите данные для входа по **FTP** (хост, логин, пароль) или используйте **файловый менеджер** в cPanel.
2. Залейте в каталог **public_html** (или **www**, как указано в инструкции хостера) все файлы проекта:
   - `index.html`, `book-now.html`, `discover.html`
   - `styles.css`, `script.js`, `book-now.js`
   - папку **Images** целиком (со всеми файлами внутри)
3. Домен обычно уже привязан к хостингу при заказе. Если нет — в панели домена укажите **A-запись** на IP вашего хостинга (его даёт хостинг-провайдер).

## Что понадобится от тебя

- **Домен** — например `tengriyurt.com` (как у тебя записан).
- **Где куплен домен** — чтобы подсказать, в каком разделе искать DNS (часто «Управление доменом» → «DNS-записи» или «Зона DNS»).
- **Есть ли хостинг** — если да, напиши у кого (Beget, reg.ru, и т.д.), дам точные шаги под их панель.

После этого можно пройти шаги по выкладке и подключению домена по этой инструкции.
