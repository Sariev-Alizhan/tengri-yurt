# 🏕️ Yurta - Традиционные юрты для современного мира

Премиум-сайт для **Tengri Camp** - производство и продажа традиционных юрт по всему миру. Дизайн вдохновлен современными минималистичными трендами 2025-2026.

## ✨ Особенности дизайна

### 🎨 Минималистичный премиум стиль
- **Крупная выразительная типографика** (Playfair Display + Inter)
- **Черно-белая палитра** с акцентами
- **Курсив для выделения** ключевых слов
- **Много пространства** (whitespace) между секциями
- **Чистые линии** и границы

### 📱 Интерактивные элементы
- **3D глобус** - вращаемая планета с метками юрт
- **FAQ аккордеон** - раскрывающиеся вопросы/ответы
- **Адаптивная навигация** - фиксированная при скролле
- **Модальные окна** для быстрых запросов
- **Плавные анимации** при прокрутке

### 🌍 Структура сайта
1. **Hero** - полноэкранное видео степи
2. **Intro** - философия юрт
3. **Статистика** - год основания, Burning Man 2025
4. **Каталог** - 5 моделей с ценами
5. **3D Глобус** - география проектов
6. **Опыт** - что ждёт гостей

8. **FAQ** - частые вопросы
9. **Контакты** - форма и данные

## 📝 Кастомизация

### 1. Добавить реальные цены
В `index.html` найдите:
```html
<div class="catalog-price">Цена по запросу</div>
```
И замените на реальную цену, например:
```html
<div class="catalog-price">от $X,XXX</div>
```

### 2. Добавить локацию на глобус

**В `script.js`:**
```javascript
{
    lat: 50.7933,
    lng: 75.6906,
    name: 'Название',
    country: 'Страна',
    description: 'Описание',
    year: '2024',
    count: '10 юрт'
}
```

**В `index.html`:**
```html
<button class="location-card" data-lat="50.7933" data-lng="75.6906">
    <div class="location-flag">🇰🇿</div>
    <div class="location-info">
        <div class="location-name">Название</div>
        <div class="location-country">Страна</div>
    </div>
</button>
```

### 3. Добавить FAQ вопрос

```html
<div class="faq-item">
    <button class="faq-question">
        <span>Ваш вопрос?</span>
        <span class="faq-icon">+</span>
    </button>
    <div class="faq-answer">
        <p>Ответ на вопрос...</p>
    </div>
</div>
```

### 4. Изменить видео-фон

Замените URL в `index.html`:
```html
<source src="ВАШ_URL.mp4" type="video/mp4">
```

**Рекомендуемые источники:**
- [Mixkit](https://mixkit.co/free-stock-video/) - "yurt", "desert", "steppe"
- [Pexels](https://www.pexels.com/videos/)
- [Pixabay](https://pixabay.com/videos/)

### 5. Настроить отправку форм

**Вариант A: EmailJS**
```javascript
contactForm.addEventListener('submit', async (e) => {
    e.preventDefault();
    emailjs.sendForm('SERVICE_ID', 'TEMPLATE_ID', contactForm)
        .then(() => alert('✅ Отправлено!'));
});
```

**Вариант B: Formspree**
```html
<form action="https://formspree.io/f/YOUR_ID" method="POST">
```

## ❓ FAQ по сайту

### Глобус не загружается?
1. Откройте консоль (F12)
2. Проверьте ошибки
3. Убедитесь в стабильном интернет-соединении (текстуры грузятся с CDN)

### Видео не воспроизводится?
1. Проверьте URL видео
2. Добавьте fallback:
```html
<video poster="fallback.jpg" ...>
```

### FAQ не раскрывается?
Проверьте, что `script.js` подключен в конце `<body>`.

### Форма не отправляется?
Откройте консоль (F12), проверьте ошибки. Для production подключите EmailJS или Formspree.
