# Tengri Yurt — Design System Reference

## Source
Generated via Stitch MCP, April 2026. Two reference screens.

## Color Palette
- **Primary Background**: #1A1510 (dark earth)
- **Deep Black**: #0F0D0A (footer, CTA sections)
- **Beige Accent**: #A89578 (Craft section only — contrast break)
- **Gold**: #C9A86E (eyebrows, dividers, prices, CTA borders)
- **White**: rgba(255,255,255, 0.6-0.95) for text
- **Green**: #25D366 (WhatsApp only)

## Typography
- **Display**: EB Garamond, weight 300-400, sizes 36-56px mobile
- **Body**: DM Sans, weight 300-400, 14-15px
- **Eyebrows**: DM Sans, 10-11px, uppercase, tracking 0.3em, gold
- **Headlines**: negative letter-spacing (-0.02em)
- **Body**: line-height 1.7-1.8

## Layout Rules
- Sharp corners (2px max roundness)
- Grain texture overlay 3% opacity on body
- Section padding: 80-140px vertical
- Alternate backgrounds: dark → photo → beige → dark
- Never two same-tone sections in a row

## Components
- **Cards**: aspect 3:4, gradient overlay from bottom, hover scale 1.05
- **Buttons**: gold border, transparent bg, uppercase tracking, 0px radius
- **Dividers**: 1px gold lines, 60px width
- **Numbers**: EB Garamond italic, faded gold (20% opacity)
- **Photos**: full-bleed, cinematic multi-gradient overlays
